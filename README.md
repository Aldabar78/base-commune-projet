# Socle documentaire commun

Ce dossier est une **fondation documentaire réutilisable** pour de futurs projets open source.

Il ne s'agit pas d'un projet en soi, mais d'un point de départ : une structure de fichiers déjà réfléchie, déjà remplie sur les principes génériques, et clairement marquée là où elle doit être adaptée à un projet réel.

## Comment l'utiliser

1. Copier ce dossier à la racine d'un nouveau projet.
2. Lire `00_CONSTITUTION.md` et `01_PHILOSOPHIE.md` pour comprendre l'esprit du socle.
3. Parcourir chaque fichier et compléter les sections marquées :

   > À compléter pour ce projet.

4. Adapter, retirer ou simplifier ce qui ne convient pas — rien ici n'est gravé dans le marbre, sauf ce que la Constitution désigne explicitement comme non négociable.

## Ordre de lecture conseillé

1. `00_CONSTITUTION.md`
2. `01_PHILOSOPHIE.md`
3. `02_CHARTE_DEVELOPPEMENT.md`
4. `03_ARCHITECTURE.md`
5. Le reste, selon les besoins du moment.
